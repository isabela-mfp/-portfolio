VPS Control Panel
This was the first project I made as a Junior developer. I had to create a basic CRUD system with OVH services to help company's support department manage customer's VPS. It was made using Node.js and Marko in front-end. Some time later, the front-end was changed to use React. Despite being a Junior, all project was made by myself. I counted on help and supervision of the company's senior developers. They assigned smaller tasks to complete each step until the project was done. There were many challenges in this project, since it was my first contact with most technologies. However, the most complicated task was working with OVH API since it was constantly changing without warning. The part I liked the most was working with NodeJs. I liked how it is structured and organized in a way that makes sense and makes programming easy and intuitive.
