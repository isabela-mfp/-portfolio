Prototype Arena
In this project the idea of the Resource Assignment Algorithm was used to create and AI for a characters battle game. The resources to be allocated were mapped to the skills of each character. The first part was focused in a creation of a game playable by two human players. The second part focused on a design for the AI player using the Resource Assignment Algorithm. Both parts were made using the open source game engine Godot, in GDScript, it's own script based in Python. As a college final paper, I made the all the project alone. The main challenges of this project were the game design of the base game, that could not be complex, and the mapping of the Resource Assignment Algorithm, created to work with 4X turn-based strategy game. Despite the challenge, the IA creation was my favorite part because I applied what I learned and saw in pratice how a simple algorithm can act like a real player.
