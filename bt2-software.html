BT2 Software
BT2 is a software to be used in Neurofeedback training sessions. It processes brain signals and provides a variety of feedback for a complementary and integrative therapeutic approach fo mental health. The software was made using QtCreator IDE and C++ language. It is structured in designs, each one with objects that will mesure brain signals and adapt feedback types to respond to the treatment proposed for each patient. BT2 was made by a team of trainers and developers which ensured that the program not only had well-designed code, but also took into account the experience of the user-trainer. The challenges of this project were the IDE's specific features since it adapted most of the C++ types and functions. The part I liked the most was work with a structured project that is made of little modules that make up the whole system.
